# COMP3000 - CHEROPHOBIA

[POSTER IMAGE]

## Project/Game Description
Cherophobia is a survival-horror in which you must avoid happiness sources at all costs - Happiness is a Sin, Nothing lasts forever. Explore a weird yet familiar environment, avoid a multitude of threats including Happy Memories, Traps and Enemies roaming the place, whilst completing challenging puzzles that will slowly progress your escape through a Master Door. Do you have what it takes?

## Features
- Dynamic gameplay and visual/sound effects based on current player status
- An unique take on Character Sanity (Happiness Meter), inspired by the likes of Amnesia™.
- Multiple encounters within the gameplay environment that challenge the player and rise tension (Roaming Enemies, Happy Memories, and Traps)
- A multitude of puzzles and challenges to complete in order to obtain items to slowly unlock your escape via the Master Door
- And more!

## Technologies used
- Game Engine: Unity version 2022.3.50f1
- Programming Language: C#
- IDE: Visual Studio Community 2022
- 3D Assets: Blender 4.5
- Image Editting: GIMP 3.0.8-2
